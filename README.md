# Overloading operators

I "re-invented" a class of fixed point numbers with overloaded operators for it.

---

### System Requirements

- Linux (Ubuntu)
- g++
- make
- VSCode

---

### Explore
You can explore my solution with VSCode or any other suitable IDE. 

### Build

```bash
git clone https://github.com/romolog/CPP__under_the_hood__overloading_operators_2025.git fixed;
cd fixed;
make;
```

### Run
```bash
./fxp;
```
